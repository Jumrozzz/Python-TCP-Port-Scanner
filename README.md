# PyScanner - Multithreaded Port Scanner 🛡️

PyScanner ek fast aur efficient Python-based tool hai jo target IP address par common ports ki status (Open/Closed) check karne ke liye banaya gaya hai. Ye tool cybersecurity enthusiast aur ethical hackers ke liye network reconnaissance mein madadgar sabit ho sakta hai.


## ✨ Features
- **Fast Scanning:** Multithreading support ke saath fast results.
- **Service Detection:** Common ports (HTTP, SSH, FTP, etc.) ke liye automatic service identification.
- **Clean UI:** Organised table format aur color-coded output.
- **Efficient:** Queue-based task management ka istemal.
